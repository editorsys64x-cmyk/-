📜 Правила

📌 Общие положения

  • Турниры проводятся по системе плей-офф (олимпийская сетка).
  • Каждый матч длится фиксированное время.

🗳️ Голосование

  • Голосовать может только авторизованный пользователь.
  • Один пользователь — один голос за матч.

👥 Участники турнира

  • Участники (персонажи, статьи, изображения) добавляются администратором перед стартом.
  • Минимум 2 участника для создания турнира.

💬 Чат и комментарии

  • Общий чат доступен всем авторизованным пользователям.
  • Запрещены: спам, оскорбления, флуд, NSFW-контент.

🔐 Авторизация

  • Доступны два способа входа: через email/пароль (Supabase) и через Fandom Wiki.
  • При входе через Fandom требуется подтверждение владения аккаунтом.

⚙️ Администрирование

  • Администратор может создавать, запускать и завершать турниры.
  • Администратор может принудительно завершить матч или назначить победителя.

🏆 Определение победителя

  • Победитель турнира — участник, выигравший финальный матч.
  • Результаты всех турниров сохраняются в истории.
